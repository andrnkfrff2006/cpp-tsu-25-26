В соответствии с заданием был выбран текстовый формат описания данных `.env`.
Это стандартный формат для хранения переменных окружения и конфигураций.

[*Описание формата*](https://smartmob-rfc.readthedocs.io/en/latest/2-dotenv.html)

### Структура файла
Данные хранятся построчно в виде пар `KEY=VALUE`.
Пример того, как программа сохраняет данные:

```env
MAX_CONNECTIONS=100
SERVER_PORT=8080
TIMEOUT_MS=5000
```
